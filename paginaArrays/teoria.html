Teoría sobre arrays

Formas de declarar un array

Utilizando el constructor new Array():

var array = new Array();

También podemos precisar el tamaño del array pasándole un parámetro al constructor: 

var array = new Array(20);
var otroArray = new Array(34);

Con esto creamos un array de 20 casillas y otro de 34 casillas.

Otra forma de utilizar el constructor es creando un "array denso", de la siguiente manera:

var array = new Array("Primer elemento", "Segundo elemento", "Tercer elemento", ...);

De forma literal

También se pueden declarar arrays utilizando los corchetes [] en lugar de la llamada a new Array(), definiendo el array de forma literal:

var array = ["Primer elemento", "Segundo elemento", "Tercer elemento" , ...];

Otra manera de definir un array de forma literal es creando un array mixto, en el que las posiciones son referenciadas con índices de tipo texto o numéricos, o ambos, de la siguiente manera:

var array = {"numero": 42, "mes": "junio", indice3: "valor", ...};

Tipos de arrays

Existen los siguientes tipos de arrays:

Paralelos:
Aquellos arrays que utilizan el mismo índice para referirse a términos homólogos, se recorren de forma sincronizada y deben tener la misma longitud para que sean homogéneos. Ejemplo: 
var profesores = ["Jose", "Jaime", "Lourdes"];
var asignaturas = ["DWES", "DIW", "DWEC"];
var alumnos = [20, 20, 22];
Al recorrerse utilizando el mismo índice a la vez, un ejemplo del resultado sería:
Jose imparte la asignatura DWES, que tiene 20 alumnos.
Jaime imparte la asignatura DIW, que tiene 20 alumnos.
Lourdes imparte la asignatura DWEC, que tiene 22 alumnos.
Unidimensionales:
Todos los arrays en JavaScript son de este tipo, aunque puedan simularse otros tipos. Ejemplo: 
var array = new Array();
array[0] = "Primer elemento";
array[1] = "Segundo elemento";
...
Para acceder a sus valores se utiliza una referencia, por ejemplo, array[1] devolvería "Segundo elemento".
Multidimensionales:
Son arrays unidimensionales que contienen otros objetos en sus posiciones, simulando así arrays de varias dimensiones. Ejemplo de un array bidimensional: 
var datos = [
["Jose", "DWES", 20],
["Jaime", "DIW", 20];
["Lourdes", "DWEC", 22];
];
Para acceder a los datos de este tipo arrays es necesario que hagamos una doble/triple/n/referencia, por ejemplo datos[2][1] devolvería "DWEC" como resultado.
Mixtos:
Sus posiciones índice están definidas por un texto o número, o ambos. Por ejemplo: 
var datos = {"numero": 42, "mes": "junio", indice3: "valor", ...};
Para acceder al contenido de una de las posiciones se utiliza el nombre dado al índice, por ejemplo datos["mes"] devolvería "junio" como resultado.

Añadir elementos a un array

Podemos hacerlo de diversas maneras:

Por asignación

La forma más convencional, tan simple como crear sentencias de asignación a cada posición del array. Por ejemplo:

var numeros = new Array();
numeros[0] = "Uno";
numeros[1] = "Dos";
numeros[3] = "Tres";
...

En la declaración

Podemos añadir elementos a la vez que declaramos el array:

var numeros = new Array("Uno", "Dos", "Tres");

O podemos prescindir del constructor:

var numeros = ["Uno", "Dos", "Tres"];

En el caso de los arrays mixtos, los datos se introducen junto al nombre de la referencia a su posición:

var numeros = {"Uno": 1, "Dos": 2, "Tres": 3};

Podríamos combinar los anterior con el uso de bucles en caso de que los valores almacenamos sean incrementos del valor anterior.

Recorrer un array

Existen diversas formas de recorrer un array

Utilizando un bucle for

Combinando un bucle for y la propiedad length, podemos desplazarnos por la longitud del array. Por ejemplo:

for(var i = 0; i < numeros.length; i++) {
    document.write(numeros[i]) + " ";
}

Utilizando un bucle while

Podemos obtener el mismo resultado combinando un bucle while y el uso de length. Por ejemplo:

var contador = 0;
while (contador < numeros.length){
    document.write(numeros[i]) + " ";
    contador++;
}

Obtendríamos también Uno Dos Tres como resultado.

Utilizando el bucle for each...in

Este método ha sido sustituido por for ... in. Su uso es el siguiente:

for each(var numero in numeros)
    document.write(numero + " ");

Utilizando el bucle for...in

Es el sustituto del anterior. Su uso es el siguiente:

for (numero in numeros) {
    document.write(numeros[numero])) + " ";
}

En todos los casos anteriores obtendríamos como resultado Uno Dos Tres.

Eliminar elementos de un array

Existen varias formas de eliminar un elemento de un array:

Ajustando el valor a null o a una cadena vacía ""

Esta forma de eliminar valores es muy antigua y debe ser evitada a toda costa, pues existen otros métodos más efectivos y eficientes, mencionados a continuación. Un ejemplo sería:

numeros[1] = "";

Empleando esta forma no se elimina completamente la posición del array.

Con el operador delete

Este operador borra el índice del elemento en el array, sin embargo, no reduce la longitud del array en sí, por lo que la memoria no se libera hasta que el intérprete JavaScript lo considere necesario.

Este operador se recomienda para arrays que usen texto como índice.

Su uso es el siguiente:

numeros.length; // Devuelve 3
delete numeros[1]; // Se elimina "Dos"
numeros.length; // Devuelve 3 otra vez
alert(numeros[1]); // Muestra "undefined"

Puede verse como no se modifica la longitud del array con el ejemplo anterior.

Con el método splice(índice, cantidad de elementos a eliminar,[elemento1 a añadir,elemento2 a añadir,...])

Este es el método más adecuado y recomendado para eliminar elementos de un array y se puede utilizar en cualquier array. Añade o elimina un elemento o secuencia de elementos de un array, reajustando la longitud de éste tras la operación.

Su uso es el siguiente:

numeros.length; // Devuelve 3
numeros.splice(1,1); // En el índice 1, eliminamos un elemento
Ahora el contenido de numeros es "Uno" "Tres"
numeros.length; // Devuelve 2

Métodos

Propiedad Descripción
constructor Devuelve la función que creó el prototipo del objeto array.
length Ajusta o devuelve el número de elementos de un array
prototype Permite añadir propiedades y métodos a un objeto

shift() Elimina el primer elemento de un array, y lo devuelve
